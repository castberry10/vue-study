# 뷰 프로젝트 생성 방법 

vue init [type]  
1. webpack  
웹팩을 활용한 프로젝트 구성, 테스팅, 문법 검사 지원  
  
2. webpack-simple  
웹팩 최소 기능을 활용한 프로젝트 구성  
  
3. browserify  
브라우저파이을 활용한 프로젝트 구성, 테스팅, 문법 검사 지원   
  
4. browserify-simple  
브라우저파이 최소 기능을 활용한 프로젝트 구성  
  
5. simple  
최소 뷰 기능만 들어간 HTML File  
  
6. pwa  
웹팩 기반 프로그레시브 웹 앱 뷰 프로젝트
